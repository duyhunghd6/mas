# Basic Multi-Agent System Architecture

*Step-by-step guidance for online seminar of GSCfin.*

## Overview

This repository contains a demonstration of creating a basic Multi-Agent System (MAS) architecture. 

The MAS (Multi-Agent System) within this demonstration is written by **Antigravity** and runs at **Claude Code**. It serves as a practical example for the GSCfin online seminar.

## Architecture

The system demonstrates the fundamental orchestration of multiple AI subagents. The master orchestrator manages tasks and delegates them to specialized subagents within the Claude Code environment.

## Step-by-Step Guidance

### 1. Requirements
Ensure you have access to a Claude Code environment where this MAS is designed to run.

### 2. Setup
Clone this repository to your local machine:
```bash
git clone https://github.com/duyhunghd6/mas.git
cd mas
```

### 3. Execution
Follow the detailed guide provided during the GSCfin seminar to initialize the Agent Flywheel and observe the parallel interactions among the subagents. 

- Watch as the Master Orchestrator spawns specialized agents.
- Observe inter-agent data passing and state management.
