# Agent Reinforcement Fine-Tuning (RFT)

1. **Log Analysis**: Read the logs of Claude Code sessions that capture all requests and responses for each SubAgent.
2. **Alignment Check**: Analyze the actual behaviors against the intended design documents and PRDs.
3. **Self-Improvement**: Define actionable instructions or updates to the SubAgent prompts and the system architecture to make self-improvements based on the observed gaps.
